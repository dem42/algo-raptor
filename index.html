How to use the raptor?

Algorithm visualizations can be accessed through links in the navigation toolbar. Each visualization is accompanied with the source code of the algorithm, implemented in Javascript.

Using the buttons in the "General Controls" section, you can start/stop the algorithm, adjust its speed or even step through the visualization line-by-line.

The small raptor heads are clickable and when clicked expand to display additional comments and observations about what is going on.
